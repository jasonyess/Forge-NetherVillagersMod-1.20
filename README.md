
# Nether Villagers Mod - v1.0.0
A mod which enables players to obtain nether-exclusive items through villager trades in the Overworld.
Can be used when access to the Nether is disallowed due to an incompatibility, bug, or anything of the sort.

## Overview
This mod adds 3 new villager professions:
- Nether Sourcer
  - Offers valuable items, such as Blaze Rods and Nether Wart
- Nether Gatherer
  - Offers nether-exclusive blocks, such as Netherrack
- Nether Looter
  - Offers chest loot that can only be obtained from the Nether, such as Pigstep (music disc)

In addition, the mod also adds 3 blocks:
- Nether Sourcer's Table
- Nether Gatherer's Table
- Nether Looter's Table

The mod also adds 4 items, which are essential:
- Copper Shard
- Gold Shard
- Diamond Shard
- Emerald Shard

## Installation
If the mod cannot be found through Curseforge's search feature, it can be installed manually.

To download the .jar file, simply direct yourself to the mod's project page and click "Download".
Project page: https://www.curseforge.com/minecraft/mc-mods/nether-villagers

Next, drag the downloaded .jar file into your mods folder, where all of your other mods are housed.

## Trades
Each Villager levels up at the same rate:
Level 1 > Level 2
 - 2 completed Level 1 trades

Level 2 > Level 3
 - 12 completed Level 1 trades
 - 3 completed Level 2 trades

No new trades are unlocked past Level 3.

The Nether Looter and Nether Sourcer villagers only accept Emerald Shards, while the Nether Gatherer only accepts emeralds.
